# NeXCon – Student Productivity App

NeXCon is a SwiftUI-based productivity app designed to help students manage daily tasks, track progress, and stay motivated.

## Features

* Task management (add, complete, delete)
* Daily motivation quotes
* Progress tracking
* Streak tracking
* Smooth animations and UI

## Tech Stack

* SwiftUI
* MVVM Architecture
* Xcode

## Future Improvements

* Cloud sync
* Notifications
* Multi-screen navigation
